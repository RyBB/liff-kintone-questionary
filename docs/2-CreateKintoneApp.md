# kintoneアプリの準備

## アプリファイルのダウンロード

[こちら](https://github.com/RyBB/liff-kintone-questionary/releases/tag/1.0.0)より **kintoneApp.zip** というZipファイルをダウンロードしてください。

## アプリの展開

kintoneにログイン後、右側のアプリの+ボタンを選択します。

**テンプレートファイルを読み込んで作成** を選択します。
